📋 Istruzioni per l'uso:

  1. Avvia il server locale:
    python3 server.py
    oppure
    ./start_server.sh
  2. Apri http://localhost:8000/wine_manager.html nel browser
  3. I file verranno caricati automaticamente, oppure caricali manualmente usando i pulsanti sopra

⚠️ Nota: Per caricare i file JSON/CSV è necessario un server HTTP locale.
Aprire il file HTML direttamente (file://) non funzionerà a causa delle restrizioni CORS del browser.
